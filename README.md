### Teste de Caixa Branca (White Box):

O teste de caixa branca ﾃｩ um mﾃｩtodo de teste de software que se concentra na avaliaﾃｧﾃ｣o dos aspectos internos do cﾃｳdigo-fonte de um programa ou sistema.
